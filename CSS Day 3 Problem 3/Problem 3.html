Showcasing Web Font Weights

This is a paragraph with font weight 300. Lighter font weights can give a more delicate and refined feel to the text, often used for secondary information or less important content.

This is a paragraph with font weight 400. Regular font weights are typically used for body text, providing a balanced and readable appearance for most content.

This is a paragraph with font weight 700. Heavier font weights are used to emphasize important information, such as headings or key points, creating a strong visual hierarchy and drawing attention to critical content.
